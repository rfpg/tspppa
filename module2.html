How far is something from zero?

Simply put, the absolute value is the distance of a given number from zero. The absolute value of a given number is indicated using the following notation: |x| where x is a given number to be computed.

    Solution:

  • Step 1: Place 5 on your number line and then count the steps from zero.
  • Step 2: There are 5 steps to the right of zero. Therefore, |5| is 5.

Example 2: Now, let's apply what we've learned to plot absolute values on a number line.

Solution:
  • Step 1: We subtract 12 from 8 to give us a value of -4.
  • Step 2: Then, we take the absolute value of -4, which is 4.
